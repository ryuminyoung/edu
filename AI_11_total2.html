이상치(Outlier) 분류 기반 집중 학습자 클리닉

이 데이터셋에서 발견되는 4가지 극단적 이상치는 표준 학습 모델에서 벗어나 있어, 단순히 '평균적 수업'을 제공할 경우 반드시 이탈하거나 방치될 대상입니다. 각 특성에 맞는 교육 설계적 혁신 솔루션을 제시합니다.

Type A: 학습 자극 부족 (지루함군)

S03: 선행형 언더스티뮬레이션 아웃라이어

성적 상 / 참여 극하

"이미 학원에서 배운 내용이라 막힘없이 풀었습니다." (수강 완료율 5%, 총 체류시간 1분, 시험 점수 100점 만점)

진단(Pedagogical Diagnosis): 지적 수준에 비해 제공되는 학습 자료의 자극 강도가 너무 낮아 학습 몰입감이 극히 저하된 상태입니다. 방치될 경우 무단 이탈이나 교실 내 무력 행사를 유발할 수 있습니다.

처방(Intervention Strategy): 교육과정 압축(Compacting)을 적용하고 심화 개별 탐구과제, 프로젝트 설계 기회, 혹은 급우 튜터링 역할을 부여하여 고차원적 인지적 자극을 공급합니다.

Type B: 고불안 / 완벽주의형 과적합

S10: 수행 불안 및 극단적 노력 과적합

스트레스 5.0 / 평가 극단

"9분 넘게 붙잡고 고민했는데 결국 틀렸네요. 자존심 상해요." (총 시험 시간 12분 중 5번에만 9분 사용, 오답)

진단(Pedagogical Diagnosis): 학업적 흥미도와 참여도(3 visits, 40m)는 우수하지만, 극단적인 완벽주의 성향으로 인해 작은 실패에 격한 정서적 충격(자존감 상실)을 느끼는 초고위험 정서군입니다.

처방(Intervention Strategy): '정답 여부'보다 '시도와 개선 과정'을 칭찬하는 과정 중심 평가를 제공합니다. 또한 '타임박스 기법'을 도입하여 무모한 몰입을 방지하고 실패를 성장의 발판으로 받아들일 수 있는 심리적 안전감을 마련해 줍니다.

Type C: 자포자기형 과제 회피

S05: 능동적 학업 거부자 (초속 찍기)

흥미 최저 / 성치 극하

"ㅇㅇㅇㅇㅇㅇㅇㅇㅇㅇㅇㅇ" (시험 총 소요 시간 단 25초, 서술형 의미 불명 타이핑, 20점)

진단(Pedagogical Diagnosis): 자아 효능감이 바닥을 치는 상태에서 문제를 풀지 않고 찍어 넘김으로써 실패에 대한 정서적 충격을 원천 봉쇄하는 '전형적 회피성 무기력' 반응입니다.

처방(Intervention Strategy): 실패할 수 없는 극단적으로 쉬운 한 단계(Micro-step)의 학습 성공 경험을 맛보여야 합니다. 텍스트 위주 대신 흥미로운 영상이나 게이미피케이션 요소를 도입하여 성취 자극을 조금씩 회복시킵니다.

Type D: 은밀한 유령 학습자 (이탈군)

S08: 장기 체류 비활동성 수동 탈락자

시간 극대 / 상호작용 제로

"모르겠음" (LMS 체류 시간 95분 최장, 동영상 완료율 10%, 클릭수 1회, 20점)

진단(Pedagogical Diagnosis): 화면만 켜둔 채 인지적으로 완전히 교실을 떠난 상태(Passive Disengaged)입니다. 장시간 온라인 플랫폼에 로그인되어 있어 정밀 행동 패턴(클릭 수)을 추출하기 전엔 교사가 파악하기 가장 힘든 수동적 이탈 상태입니다.

처방(Intervention Strategy): 플랫폼 내 실시간 상호작용 질문 팝업이나 중간 퀴즈를 탑재하여 오프라인 이탈을 원천 감지해야 합니다. 오프라인과 연계하여 1:1 상담 및 오프라인 조별 활동의 저난이도 단순 역할 부여가 권장됩니다.
